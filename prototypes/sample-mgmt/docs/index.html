产品概述

样片管理系统（Sample Management System） 是为元器件授权分销行业打造的全流程样片管理平台。系统覆盖样片申请、审批、跟踪、数据分析、系统配置和AI异常检测六大核心模块，帮助销售人员和FAE工程师高效管理样片业务。

核心价值

• 🚀 提效：4种申请模式适配不同场景，向导/批量/单页/复制历史
• 🔍 透明：全流程状态跟踪 + 实时看板 + 时间轴
• 🛡️ 风控：AI异常检测 + 多级审批规则自动匹配
• 📊 洞察：5种可视化图表 + Excel导出 + 审批时效统计

技术架构

纯前端 HTML/CSS/JS 单页应用 · 无框架依赖 · CSS Custom Properties 设计系统 · Chart.js 图表 · localStorage 持久化 · 响应式适配 Desktop/Tablet/Mobile

如何开始

样片申请模块

提供 4 种申请模式，适配不同业务场景：

1️⃣ 向导式申请（推荐）

5 步引导流程：基本信息 → 型号选择 → 数量确认 → 上传附件 → 提交预览。数据跨步骤保持，适合首次使用或多型号申请。

  1. 填写客户名称、项目等基本信息
  2. 选择型号（支持搜索过滤），可添加多行
  3. 确认各型号数量，MPQ自动校验
  4. 上传附件（规格书、测试报告）
  5. 预览提交信息，确认后提交

2️⃣ 单页式申请

所有字段在一个页面完成，适合快速申请。支持即时校验和行级管理。

3️⃣ 批量申请

表格形式多型号同时录入，支持下载Excel模板和导入CSV数据。表格包含型号/数量/原厂/MPQ/CY/交期/用途等完整字段。

4️⃣ 复制历史

浏览历史申请记录，支持按状态筛选和关键词搜索。一键复制历史申请，编辑后重新提交，减少重复录入。

审批管理模块

支持 多级审批流程，根据金额/数量/原厂类型自动匹配审批路径。

审批列表

4 个 Tab：待审批（含超期高亮）/已审批/我发起的/审批历史。待办卡片显示申请信息和审批路径。

审批操作

• ✅ 批准：一键通过，流转至下一审批节点
• ❌ 拒绝：需填写拒绝原因
• ➕ 加签：选择加签人，填写原因，新增审批节点
• ↻ 转交：选择接收人，填写说明（循环检测）

审批规则

按金额（<1万/1-10万/>10万）、按数量（<100/100-1000/>1000）、按原厂类型（欧美/日韩/大陆）触发不同审批层级，金额优先。

审批时效统计：

平均 18.5h阈值 24h超期 2 件

样片跟踪模块

提供 看板视图 和 时间轴 两种方式跟踪样片全生命周期。

看板视图

5 列看板：待审批 → 审批中 → 备货中 → 已发货 → 已签收。支持看板/列表视图切换。

时间轴

6 个关键节点：提交申请 → PM审批 → 仓库备货 → 已发货 → 客户签收 → 完成闭环。每个节点可展开查看详情。

配套页面

• SB-01 登录页：角色切换演示
• SB-02 个人中心：个人信息编辑 + 偏好设置（通知开关）
• SB-03 通知中心：消息列表，支持标记已读
• SB-04 库存概览：18 种型号库存状态卡片

数据分析看板

4 个页面 + 5 种图表 + Excel 导出，全面覆盖业务数据监控需求。

DA-01 数据总览

4 个 KPI 卡片（申请总数/待审批/通过率/平均周期）+ 申请趋势折线图（支持周/月/季/年切换）

DA-02 按型号统计

柱状图/饼图切换，展示各型号申请量Top。支持排序切换。

DA-03 按时间统计

月度（默认）/季度/年度三种粒度切换，申请数+通过数对比柱状图。

DA-04 按申请员统计

横向柱状图 + 排名表格（姓名/次数/通过率）。带导出按钮。

附加图表（增强版）

• 审批效率：通过率饼图 + 各PM平均时长柱状图
• 库存需求对比：双轴折线图
• 运营成本分析：部门维度堆叠柱状图

AI异常检测模块

模拟 5 条异常规则 的智能检测引擎，自动识别高风险申请。

异常判断规则

• 🔴 频繁申请：同一申请人30天内申请超阈值
• 🟡 同型号累积异常：同型号30天总量超阈值
• 🔴 审批时长异常短：大额申请审批<1小时
• 🟡 客户重复申请：同客户短时间多次申请
• 🟢 型号需求突变：月申请量环比增长>200%

操作流程

1. 异常列表查看风险等级和规则命中情况
2. 点击进入详情页查看AI分析结论和关联订单
3. 操作区：标记处理 / 忽略异常 / 人工复核

系统设置模块

10 个子页面 覆盖全部运维管理功能。

用户/角色管理

用户列表增删改查、角色权限矩阵（✅/❌/👁）、用户-角色分配下拉

审批规则配置

自动审批（低额）/超时自动转交/紧急绿色通道 三条规则，支持启用/停用开关

审批流程设计器

可视化流程画布：开始 → 销售提交 → PM审批 → 财务审核 → 仓库备货 → 结束。节点拖拽展示。

ERP对接模拟模块

模拟 3 种 ERP 对接场景 + 50+ 业务字段分类管理。

ERP 模拟查询

• 📦 库存查询：选择型号 → 模拟API调用（Loading动画）→ 返回库存/MPQ/仓库位置
• 📄 物料信息拉取：返回原厂/封装/规格书等完整参数
• ✅ MPQ校验：输入数量自动校验是否满足最小包装量，非整数倍时提示错误

字段分类

5 个分类标签切换：申请信息(9)/审批信息(5)/发货信息(6)/财务信息(4)/系统信息(4)。每字段展示名称/类型/必填/默认值，支持拖拽排序。

常见问题

Q: 如何切换用户角色？

A: 点击右上角用户头像 → 下拉菜单选择其他角色。也可在登录页直接选择。

Q: 申请被驳回了怎么办？

A: 查看驳回原因，修改后重新提交。可在"复制历史"中快速复用上次数据。

Q: 如何查看样片当前状态？

A: 进入"样片跟踪"模块，看板视图展示全部样片状态分布，时间轴展示单个样片完整流转。

Q: 审批超时阈值是多少？

A: 默认 24 小时。可在系统设置 > 全局参数中修改，超期申请在列表中以红色高亮。

Q: 如何导出数据看板？

A: 在 DA-04 页面点击"导出"按钮，浏览器将下载 CSV 格式数据文件。

技术说明

项目结构

prototypes/sample-mgmt/
├── index.html          ← 主入口（角色登录+SPA导航）
├── css/                ← 设计系统+组件+布局+各模块样式
├── js/                 ← 权限引擎+表单验证+申请逻辑+审批流程+图表+AI检测+ERP API
├── data/               ← 角色配置+审批规则
└── pages/              ← 12个功能页面
    ├── apply-wizard/single/batch/history.html  (4种申请)
    ├── approval-list.html          (审批管理)
    ├── tracking.html               (跟踪+公共页面)
    ├── dashboard.html              (数据分析)
    ├── ai-detection.html           (AI异常检测)
    ├── settings.html               (系统设置)
    └── erp-simulator.html          (ERP对接)

技术栈

• 纯原生 HTML/CSS/JS — 零框架依赖
• CSS Custom Properties 设计系统（60+ 设计令牌）
• Chart.js 4.4.1 CDN — 5+ 图表类型
• localStorage — 登录状态持久化
• Hash-based SPA 路由 — 模块间无刷新切换
• iframe 嵌入 — 各模块独立运行互不干扰
• 响应式布局 — CSS Grid + Media Queries
